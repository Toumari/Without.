# without.

> gentle tracking for gentle minds

A minimal habit tracker focused on what you're avoiding rather than what you're doing. Track days without a habit, celebrate milestones, and reset without judgment when you slip.

## Features

- Track multiple habits with a name, emoji, and start date
- Animated day counter with milestone badges (1 week, 1 month, 100 days, etc.)
- "I slipped" flow that resets your count gently
- Edit and delete habits
- Data stored locally on your device — no account required
- Installable as a PWA

## Stack

- [React 19](https://react.dev/) + [TypeScript](https://www.typescriptlang.org/)
- [Vite](https://vitejs.dev/)
- [Tailwind CSS v4](https://tailwindcss.com/)
- [Framer Motion](https://www.framer.com/motion/)
- [vite-plugin-pwa](https://vite-pwa-org.netlify.app/)

## Getting started

```bash
npm install
npm run dev
```
